# Development Notes

## Enable developer mode

1. Open the Scene State panel.
2. Enable **Developer mode** in settings.
3. The Developer section becomes visible.

## Run fixtures

1. Enable developer mode.
2. Click **Run Fixtures**.
3. Review the summary in the Developer section.

## Verify pins and locks

1. Generate a snapshot.
2. Open **Characters**.
3. Toggle **Pin** or **Lock posture/support**.
4. Send new messages and confirm pinned entities remain and locks resist changes unless explicit contradictions appear.

## Test export downloads

1. Generate a snapshot.
2. In **Canonical YAML**, click **Download YAML**.
3. Confirm the file saves with the expected name.
